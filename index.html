Orígenes y Herramientas de la Web

I. Orígenes de la Web

  • 1. Web 2.0

    La Web 2.0 se caracteriza por la participación activa de los usuarios. En lugar de solo consumir contenido, los usuarios pueden crear y compartir contenido en plataformas como redes sociales y wikis.

  • 2. Web 3.0

    La Web 3.0, también conocida como Web Semántica, busca hacer que la información en la web sea más comprensible para las máquinas. Utiliza tecnologías como RDF y OWL para mejorar la interoperabilidad de datos.

  • 3. La W3C y los estándares

    La World Wide Web Consortium (W3C) es una organización que desarrolla estándares para asegurar la interoperabilidad y la calidad de la web. Incluye estándares para HTML, CSS, y más.

II. Lenguajes de etiquetas

  • 1. SGML

    El Standard Generalized Markup Language (SGML) es un lenguaje de marcado que define la estructura y el contenido de documentos. Es el precursor de XML y HTML.

  • 2. XML

    Extensible Markup Language (XML) es un lenguaje de marcado que define reglas para el formato de documentos. Se utiliza para almacenar y transportar datos de manera estructurada y legible.

  • 3. HTML

    Hypertext Markup Language (HTML) es el lenguaje estándar para la creación de páginas web. Define la estructura del contenido a través de elementos y etiquetas.

  • 4. XHTML

    Extensible Hypertext Markup Language (XHTML) es una reformulación de HTML 4.01 usando la sintaxis de XML. Es más estricto en su estructura y presentación de documentos.

  • 5. HTML5

    HTML5 es la última versión de HTML que introduce nuevas características como video, audio, y gráficos. Mejora la semántica y la funcionalidad para aplicaciones web más avanzadas.

III. Otros lenguajes relacionados al desarrollo web

  • 1. CSS/CSS3

    CSS (Cascading Style Sheets) es un lenguaje para describir la presentación de un documento HTML. CSS3, la última versión, agrega características avanzadas como animaciones y transiciones.

  • 2. JavaScript

    JavaScript es un lenguaje de programación que permite crear interactividad en las páginas web. Es fundamental para el desarrollo de aplicaciones web dinámicas y modernas.

  • 3. Ajax

    Asynchronous JavaScript and XML (Ajax) es una técnica para crear aplicaciones web interactivas. Permite actualizar partes de una página sin recargarla completamente.

  • 4. Tecnologías y lenguajes del lado del servidor

    Estos incluyen lenguajes como PHP, Python, y Ruby, que se ejecutan en el servidor para generar contenido dinámico y manejar la lógica del backend.

  • 5. Navegadores

    Los navegadores web como Chrome, Firefox, y Safari interpretan y muestran el contenido HTML, CSS y JavaScript. Su compatibilidad con estándares puede variar.

  • 6. Motores renderizados

    Los motores renderizadores, como Blink y Gecko, son responsables de convertir el código HTML, CSS y JavaScript en una representación visual en el navegador.

IV. Herramientas de desarrollo

  • 1. Software WYSIWYG

    Los editores WYSIWYG (What You See Is What You Get) permiten diseñar páginas web visualmente sin necesidad de escribir código manualmente. Ejemplos incluyen Adobe Dreamweaver.

  • 2. Otras alternativas
    • i. Figma

      Figma es una herramienta de diseño colaborativo basada en la web, ideal para prototipos y diseño de interfaces. Permite la edición en tiempo real y colaboración entre equipos.

    • ii. InVision

      InVision es una plataforma de diseño y prototipado que facilita la creación de prototipos interactivos y la colaboración entre diseñadores y desarrolladores.

    • iii. Zeplin

      Zeplin facilita la colaboración entre diseñadores y desarrolladores proporcionando una interfaz para compartir y exportar estilos y especificaciones de diseño.

    • iv. GitHub

      GitHub es una plataforma de control de versiones y colaboración para proyectos de desarrollo. Permite la gestión de código fuente y la colaboración en equipo a través de repositorios.

Inspección de Código desde el Navegador

Inspección de Código

La inspección de código desde el navegador es una técnica esencial para desarrolladores web que permite ver y modificar el HTML, CSS y JavaScript de una página en tiempo real. Aquí se describe cómo hacerlo:

  • 1. Acceso a las Herramientas de Desarrollador

    Para inspeccionar el código de una página web, primero debes abrir las Herramientas de Desarrollador del navegador. En la mayoría de los navegadores, puedes acceder a ellas presionando F12 o Ctrl + Shift + I (en Windows) o Cmd + Option + I (en macOS).

  • 2. Inspección del HTML

    Una vez abiertas las Herramientas de Desarrollador, dirígete a la pestaña "Elements" o "Inspector". Aquí puedes ver la estructura HTML de la página. Puedes hacer clic en los elementos para ver sus propiedades y modificar el HTML directamente.

  • 3. Modificación del CSS

    En la misma herramienta, en la pestaña "Styles" o "Estilos", puedes ver y editar las reglas CSS aplicadas a los elementos seleccionados. Esto te permite ajustar el diseño visual en tiempo real.

  • 4. Depuración de JavaScript

    La pestaña "Console" o "Consola" permite ver mensajes de error y ejecutar código JavaScript directamente. También puedes usar la pestaña "Sources" para depurar scripts y establecer puntos de interrupción.

Estas herramientas son fundamentales para depurar y ajustar el código de las páginas web, facilitando el desarrollo y la resolución de problemas.
